## 🍎 Session Memory MCP - Mac 一键安装

打开终端（Terminal），复制粘贴以下命令：

### 方法一：从 GitHub 安装（推荐）

```bash
git clone https://github.com/eva-wanxin-git/session-memory-mcp.git
cd session-memory-mcp
chmod +x install-mac.sh
./install-mac.sh
```

### 方法二：从本地安装包（如果已下载）

```bash
# 解压安装包到Documents目录
unzip ~/Downloads/session-memory-mcp-Mac-安装包-*.zip -d ~/Documents/

# 进入目录
cd ~/Documents/session-memory-mcp

# 运行安装
chmod +x install-mac.sh
./install-mac.sh
```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## ✅ 安装完成后

**重要**: 完全退出并重启 Cursor

## 🧪 测试安装

在 Cursor 中说：

```
我是eva，帮我创建一个新任务
```

系统会自动使用 Session Memory 创建任务和会话！
